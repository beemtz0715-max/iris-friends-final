Iris & Friends Care Tracker
A small full‑stack application for managing daily cat care tasks and vet records.
Built with Vue 3 (frontend) and Cloudflare Workers (backend).

GitHub repo
https://github.com/beemtz0715-max/iris-friends-final

Live site (frontend)
https://iris-friends-final.pages.dev/login

Features (MVP)
User registration and login

Secure token‑based authentication

Dashboard showing all cats

Daily care tasks for each cat

Ability to mark tasks complete/incomplete

Vet record history per cat

Ability to add new vet records

Add‑a‑cat modal with color picker

Logout functionality

How to Use the App
Go to the live deployed URL
Register with any email and password
Log in
Select one of the existing cats or add your own
Test daily tasks and vet record features
Logout when finished
No special setup required for testing.

Evolution & Changes
The original idea included syncing with real vet offices.
This was removed because it was not technically possible within the project scope.
The core app idea stayed the same.

Features That Were Cut
Syncing with local vet offices
Automated vet appointment imports
External API integrations
These were removed due to time and technical limitations.

Future Improvements
Ability to sync with local vet offices
Push notifications for medication reminders
